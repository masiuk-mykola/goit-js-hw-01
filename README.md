# goit-js-hw-01

Домашнє завдання №1 з JavaScript.

## Структура проєкту

```
.
├── index.html
├── README.md
└── js/
	├── task-1.js
	├── task-2.js
	└── task-3.js
```

## Опис завдань

- `task-1.js` — функція `makeTransaction(quantity, pricePerDroid)` повертає рядок із вартістю замовлення дроїдів.
- `task-2.js` — функція `getShippingMessage(country, price, delivery)` повертає повідомлення з повною вартістю доставки.
- `task-3.js` — функція `getElementWidth(content, padding, border)` обчислює загальну ширину елемента на основі значень у `px`.

У кожному файлі є приклади виклику функцій через `console.log`.

## Як запустити

1. Відкрий `index.html` у браузері.
2. Відкрий DevTools → Console, щоб побачити результати виконання.

Альтернатива: запусти Live Server у VS Code для зручного перегляду.
